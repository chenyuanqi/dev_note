
### 运算符优先级
| 序号 | 运算符   |  描述  |
| ---- | :-----:  | :----: |
| 1  | ** | 指数(次幂)运算 |
| 2  | ~ + - | 补码，一元加减(最后两个的方法名称是+@和-@) |
| 3  | * / % // | 乘法，除法，模数和地板除 |
| 4  | + - | 加法，减法 |
| 5  | >> << | 向右和向左位移 |
| 6  | & | 按位与 |
| 7  | ^ | 按位异或和常规的“OR” |
| 8  | <= < > >= | 比较运算符 |
| 9  | <> == != | 等于运算符 |
| 10  | = %= /= //= -= += *= **= | 赋值运算符 |
| 11  | is is not | 身份运算符 |
| 12  | in not in | 成员运算符 |
| 13  | not or and | 逻辑运算符 |

> 注意：如上优先级从高到底排列。
